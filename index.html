ℹ CAPTURING CALLER AUDIO — TWO METHODS

METHOD A — Chrome Tab (browser-based calls only)

  1. Use Screen + Tab + Mic mode.
  2. In the share dialog, switch to "Chrome Tab".
  3. Select your call tab (Messenger, Meet, etc.).
  4. Check "Share tab audio" → click Share.
  5. Works for web-based calls only — NOT desktop apps.

METHOD B — VB-Cable (MS Teams, Zoom, Skype desktop)

  1. Install VB-Cable (free): vb-audio.com/Cable
  2. Windows Sound → Playback → set CABLE Input as default.
  3. In VCREC, select ★ CABLE Output in Caller Audio Device.
  4. Use Entire Screen or Window mode → START.
  5. Status bar will show LOOPBACK when active.
